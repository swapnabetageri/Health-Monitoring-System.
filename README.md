  Health Monitoring System (ESP32 IoT)

 Project Description
ESP32-based IoT Health Monitoring System for real-time monitoring of temperature, heart rate, SpO₂, humidity, and air quality.

 Features
- Real-time sensor monitoring  
- Measures temperature, heart rate, SpO₂  
- Monitors humidity and air quality  
- Wireless IoT-based data transmission  


Technologies Used
- ESP32 Microcontroller  
- Arduino IDE  
- Embedded C  
- Sensors (Pulse, DHT11, MQ Gas, SpO₂)  
- Wi-Fi Communication 

 How It Works
- Sensors collect health and environment data  
- ESP32 processes sensor data  
- Data is displayed or sent via Wi-Fi  
- Enables real-time monitoring
